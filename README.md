## **Iris Dataset Analysis (Exploratory Data Analysis)**

### **Introduction**

This project focuses on analyzing the Iris dataset using data visualization and statistical methods. The dataset contains measurements of different iris flowers, including sepal length, sepal width, petal length, and petal width for three species.

---

### **Steps Performed**

#### **1. Data Loading**

The dataset was loaded from Google Drive into a Pandas DataFrame. Column names were manually assigned because the dataset did not contain headers.

---

#### **2. Data Inspection**

Basic exploration was performed to understand the dataset:

* Shape (rows and columns)
* Column names
* First few rows (head)
* Data types and null values (info)
* Statistical summary (describe)

---

#### **3. Data Visualization**

**(a) Scatter Plots**
Two scatter plots were created:

* Sepal Length vs Sepal Width
* Petal Length vs Petal Width

These plots help visualize the relationship between features and different flower species.

---

**(b) Histograms**
Histograms with KDE curves were used to understand the distribution of each numerical feature.

---

**(c) Box Plot**
A box plot was created to detect outliers in the dataset and understand the spread of numerical features.

---

### **Conclusion**

The Iris dataset analysis provides insights into feature distributions, relationships, and possible outliers. Visualization helps in understanding how different species of flowers vary based on their physical measurements.
